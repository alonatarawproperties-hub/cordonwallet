VALIDATION RULES
EVM private key:
- Accept 64 hex characters, with or without "0x"
- Validation:
  - After trimming, if startsWith("0x") remove it
  - Must be exactly 64 hex chars and match /^[0-9a-fA-F]{64}$/
- If invalid, show inline error and disable Import button

Solana private key:
Support two common formats:
1) JSON array: "[12,34,...]" (64 numbers)
   - Detect if trimmed starts with "["
   - JSON.parse -> number[]
   - Must be length 64
   - Each number 0..255
   - Uint8Array.from(numbers)
2) Base58 secret key string (64 bytes)
   - If not JSON, treat as base58
   - Decode base58 to Uint8Array
   - Must be length 64
- Create Keypair.fromSecretKey(secretUint8)
- Address = keypair.publicKey.toBase58()

IMPORTANT:
- Do NOT accept a Solana *public* address here. This is private key import.
- Show clear error if decode fails or length not 64 bytes.
